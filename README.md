# To-Do List Application

A Django-based To-Do List application that allows users to create, view, edit, and delete tasks.

## Features
- Add new to-do items
- View to-do details
- Edit existing to-dos
- Delete to-do items

## To-Do Fields
- Subject
- Notes

## Modules
### Module 1: To-Do Creation & Listing
- Create new to-do items
- Display all to-dos on the home page

### Module 2: To-Do Update & Deletion
- View, edit, and delete to-do items

## Technologies Used
- Python
- Django
- HTML
- CSS
- Git & GitHub

python manage.py migrate
